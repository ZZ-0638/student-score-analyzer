# **学生成绩数据分析系统**



##### **项目简介**



本项目是一个基于 C 语言的控制台学生成绩管理系统，在实现基本的增删改查功能基础上，重点增加了 数据分析报告模块 ，帮助教师快速了解班级整体学习情况，识别成绩分布和潜在风险学生。系统支持批量生成随机数据，方便功能测试和数据规模演示。



##### **主要功能**



* **添加学生** ：手动输入学号、姓名、成绩，程序自动进行合法性检查（如学号为正整数，成绩在 0-100 之间）。
* **打印学生列表** ：以清晰表格形式输出所有学生信息。
* **删除学生** ：根据学号删除指定学生记录。
* **更新成绩** ：根据学号修改学生成绩。
* **插入学生** ：在指定位置插入新学生记录。
* **生成成绩报告** ：一键输出统计报告，包含总人数、平均分、最高/最低分（含姓名）、及格率（≥60）、优秀率（≥85），以及分数段分布（0-59、60-69、70-79、80-89、90-100）。
* **批量生成随机数据** ：自动生成指定数量的学生记录，学号自动递增，姓名格式为 StudentX，成绩随机 0-100，用于快速测试。



##### **技术栈**



* **语言** ：C（C99 标准）
* **开发环境** ：Dev-C++ / Visual Studio / GCC
* **核心概念** ：结构体、数组、函数模块化、标准输入输出处理



##### **运行方法**



&#x20;**1. 克隆仓库**


git clone https://github.com/你的用户名/student-score-analyzer.git



&#x20;**2. 编译**

使用 GCC 编译：

gcc main.c -o score\\\_analyzer

或在 Dev-C++ 中直接打开 `.c` 文件，点击“编译运行”。



&#x20;**3. 运行**

执行编译后的可执行文件：


./score\\\_analyzer     Linux/Mac
score\\\_analyzer.exe   Windows


&#x20;**4. 使用菜单**

程序启动后显示主菜单，按数字键选择对应功能：


1\\. Add Student
2\\. Print Students
3\\. Delete Student
4\\. Update Student
5\\. Insert Student
6\\. Generate Report
7\\. Generate Random Data
8\\. Exit

##### **设计思路**



* **数据存储** ：使用结构体数组（最大容量 100），通过全局变量 `studentCount` 记录当前学生数。
* **模块化设计** ：每个功能独立为一个函数，主函数通过循环菜单调用。
* **输入缓冲处理** ：在 `scanf` 后使用 `getchar()` 吸收换行符，避免输入错位。
* **数据分析** ：遍历数组一次完成所有统计指标的计算，时间复杂度 O(n)。
* **批量生成** ：利用 `rand()` 生成随机成绩，`sprintf` 格式化姓名，提高测试效率。
